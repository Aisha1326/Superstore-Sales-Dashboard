# Superstore-Sales-Dashboard
Superstore Mini Data Analysis

Analyzed Superstore dataset to uncover sales trends, top categories, and regional performance using Google Sheets.

Tools Used: Google Sheets (formulas, pivot tables, charts)

Key Insights:

Identified top-selling product categories and sub-categories.

Highlighted regions with highest sales and revenue patterns.

Visualized trends to support business decisions.
